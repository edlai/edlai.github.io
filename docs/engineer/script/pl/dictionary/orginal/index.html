■ 1. 系統首頁 與 查詢 (以 ARP為例) - Step. I

進入系統首頁。依照一般字典的查法，如果我們要查詢 Hello 這個字，
會先翻到 以 H 為字首的部分，再查詢字尾是以 ello 結尾的部分。

這邊的做法也是如此，如果要查詢 ARP 這個字 ，

  1. 先在主索引區 點選 A 之後 ( 圖示最右邊的 A )

  2. 次索引會出現

  3. 點選展開該下拉功能表

  4. 上圖點選 ARP 後，即會跳到 ARP 的名詞解釋區 ( 續下 )
